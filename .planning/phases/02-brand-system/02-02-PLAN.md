- Phone number and "Get Started" CTA at bottom of drawer
- Close on: hamburger toggle, overlay click, nav link click

**Vanilla JS `<script>` tag:**
- Toggle translate-x-full on menu
- Toggle opacity on overlay
- Toggle hamburger/close icons
- Set aria-expanded and aria-hidden
- Lock body scroll when open (`document.body.classList.add('overflow-hidden')`)
- Unlock on close
- Close on overlay click and nav link click

**Do NOT:**

- Add `tailwind.config.js` — Tailwind v4 uses CSS-only config
- Add gradient to header background — use solid navy per discretion
